# postcodes-nl

To build you need:

    sudo apt install 7zip wget php-cli gh

To run you need:

    sudo apt install 7zip php-cli php-mysql

Build using:

    bash release

Run using:

    bash postcodes-nl.sh

Configure your database in `postcodes-nl.php`.
